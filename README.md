## Prerequisites:
- pip install openai
- pip install fastapi
- pip install google-search-results
- pip install dotenv
  
## Usage 
To run frontend:
 - cd frontend/src
 - npm install
 - npm run 

To run backend:
- cd backend
- uvicorn main:app --reload
