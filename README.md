## Prerequisites:
```
cd backend
pip install -r requirements.txt
```

```
cd frontend/src
npm install
```
  
## Run the Application: 
To run frontend in `/frontend/src`:
```
npm run
```

To run backend in `/backend`:
```
uvicorn main:app --reload
```
